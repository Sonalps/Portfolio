# Sonal Phatige Sanjay — Portfolio

> Data Engineer · AI Systems · MS Data Science @ Boston University

A clean, single-file HTML portfolio — no frameworks, no build step, zero dependencies (except Google Fonts). Just drop `index.html` into any static host and go.

## 🚀 Live Demo

Push to GitHub and enable **GitHub Pages** (Settings → Pages → Deploy from branch `main`, folder `/root`) — your portfolio will be live at:

```
https://<your-username>.github.io/<repo-name>/
```

## 📁 Structure

```
.
└── index.html   ← entire portfolio (HTML + CSS + JS, self-contained)
```

## ✨ Features

- Animated data pipeline hero diagram (matches real HeiBuddy work)
- Skills ticker with auto-scroll
- Timeline-based experience section with scroll-reveal animations
- Project cards with quantified impact stats
- Responsive — works on mobile, tablet, desktop
- No build tools, no npm, no React

## 🛠 Customizing

All content is inline in `index.html`. To update:

| What | Where in the file |
|---|---|
| Name / tagline | `<h1>` in `#hero` |
| Contact details | `#contact` section |
| Experience bullets | `.timeline` section |
| Projects | `.projects-grid` section |
| Skills | `.skills-grid` + ticker |
| Colors | `:root` CSS variables at top |
